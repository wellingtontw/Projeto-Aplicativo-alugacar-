# 🚗 AlugaCar

Sistema completo de aluguel de veículos com:

- Backend PHP + MySQL
- API REST
- Aplicativo Flutter

## Funcionalidades
- Login
- Listagem de carros
- Aluguel
- Cancelamento
- Histórico

## Como rodar

1. Instalar XAMPP
2. Importar banco (pasta banco)
3. Configurar config.php
4. Rodar localhost
